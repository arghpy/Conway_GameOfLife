# Conway_GameOfLife

This is a personal project I've done in order to practice C and the raylib library.
Please feel free to experiment the *ROWS*, *COLS*, *width*, *height* and *POPULATION* variables as they determine the size of the model.

The project was developed on Linux and thus is suited to UNIX operating systems.

In order to compile, use `gcc` or `clang`.
E.g.: `gcc GameOfLife.c -o GameOfLife -lraylib`
